Nuestros Servicios

Res-Q es un CRM social orientado a la acción efectiva, la protección de derechos y la intervención inmediata en casos de vulnerabilidad infantil.

Principales Funcionalidades

Rol del Administrador

  • Carga de situaciones nuevas
  • Asignación de casos a usuarios operadores
  • Generación de reportes globales del sistema
  • Configuración de usuarios, contactos y notificaciones

Rol del Operador

  • Visualización de situaciones asignadas
  • Seguimiento y carga de novedades
  • Generación de reportes propios
  • Acceso a la agenda, comentarios y archivos adjuntos

Funcionalidades Destacadas

  • Autenticación con doble factor (Google Authenticator)
  • Geolocalización de casos y zonas críticas (API Google Maps)
  • Colores diferenciales por estado del caso (abierto, en seguimiento, resuelto)
  • Notificaciones automáticas (móvil y mail)
  • Reportes en formato exportable
  • Acceso web y aplicación móvil

Tecnologías Utilizadas

  • Backend: C# .NET
  • Frontend Web: JavaScript
  • Aplicación Móvil: APK nativa
  • Base de Datos: Microsoft SQL Server
  • APIs: Google Maps, autenticación
  • DevOps: Azure DevOps

¿A quién está dirigido?

  • Trabajadoras/es sociales
  • Municipios e instituciones públicas
  • Áreas de infancia y desarrollo social
  • Organismos que atienden casos de vulnerabilidad
  • ONGs y redes territoriales

Objetivos del Sistema

  • Mejorar la eficiencia en el registro, seguimiento y resolución de casos
  • Centralizar la información relevante en una única plataforma accesible y segura
  • Facilitar la comunicación entre las distintas áreas involucradas
  • Promover la acción temprana mediante recordatorios y alertas automatizadas
  • Brindar una herramienta amigable e intuitiva para equipos interdisciplinarios

Beneficios del Sistema

Eficiencia Operativa

  • Automatización de procesos rutinarios
  • Reducción de tiempos administrativos
  • Optimización de recursos humanos

Mejora en la Atención

  • Seguimiento continuo de casos
  • Alertas tempranas de situaciones críticas
  • Historial completo de intervenciones

Comunicación Institucional

  • Coordinación entre diferentes áreas
  • Transparencia en los procesos
  • Trazabilidad completa de acciones

Análisis y Reportes

  • Informes estadísticos en tiempo real
  • Identificación de patrones y tendencias
  • Datos para toma de decisiones

Versiones Disponibles

Versión Web

Acceso completo desde cualquier navegador web, ideal para trabajo de oficina y gestión administrativa.

Aplicación Móvil

APK para dispositivos Android, perfecta para trabajo de campo y acceso inmediato a la información.

Modelo de Sostenibilidad

Ofrecemos diferentes paquetes adaptados a las necesidades de cada institución:

  • Implementación: Instalación y configuración inicial del sistema
  • Capacitación: Formación completa para usuarios y administradores
  • Soporte técnico: Asistencia continua y mantenimiento
  • Personalización: Adaptación a necesidades específicas de cada municipio
  • Actualizaciones: Mejoras continuas y nuevas funcionalidades
